# Stremio for Google

A Chrome extension that adds a **Stremio** button to Google's "Where to Watch" and "Watch show" panels when searching for movies and TV shows.

## Installation

### From Source (Developer Mode)

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the extension folder

### Requirements

- Google Chrome (or Chromium-based browser)
- [Stremio](https://www.stremio.com/) desktop app installed

## How It Works

1. Search for any movie or TV show on Google
2. Look for the "Where to Watch" or "Watch show" section
3. Click the purple **Stremio** button
4. The content opens directly in your Stremio app

The extension automatically detects the IMDb ID from Google's search results and opens the correct content in Stremio.
