# Stremio for Search

A Chrome extension that adds a **Stremio** button to Google and DuckDuckGo search results for movies and TV shows.

## Installation

### From GitHub Release (Recommended)

1. Go to the [latest release](https://github.com/Sean-Ryan6423/Open-In-Stremio/releases/latest)
2. Download `Open-In-Stremio.zip`
3. Extract the zip file to a folder
4. Open Chrome and go to `chrome://extensions/`
5. Enable **Developer mode** (toggle in top right)
6. Click **Load unpacked**
7. Select the extracted folder

### From Source

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the extension folder

### Requirements

- Google Chrome (or Chromium-based browser)
- [Stremio](https://www.stremio.com/) desktop app installed

## How It Works

1. Search for any movie or TV show on Google or DuckDuckGo
2. Look for the "Where to Watch" section (Google) or movie info box (DuckDuckGo)
3. Click the purple **Stremio** button
4. The content opens directly in your Stremio app

The extension automatically detects the IMDb ID from search results and opens the correct content in Stremio.
